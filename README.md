# ✦ Shelf Esteem ✦

> *Tell us what you've read. We'll listen. We'll judge.*

Shelf Esteem is an AI-powered literary taste analyzer that delivers witty roasts, personality archetypes, compatibility warnings, and book recommendations based on your reading habits.

Because your bookshelf says more about you than your résumé ever could.

---

## ✨ Features

*  Live book search using Google Books API
*  Build your shelf with up to 8 books
*  AI-generated literary roasts (Shelf Assessment)
*  Personality archetypes and trait analysis
*  Compatibility warnings for anyone brave enough to date you
*  Personalized book recommendations
*  Dark and light themes
*  Interactive dot background with ripple effects
*  One-click result sharing

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/shelf-esteem.git
cd shelf-esteem
```

Install dependencies:

```bash
npm install
```

Create a `.env.local` file:

```env
GROQ_API_KEY=your_api_key_here 
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
You will see the websie run in your system!

## 🧠 How It Works

1. Search and add books to your shelf.
2. Shelf Esteem sends your reading choices to Illama.
3. The Oracle interprets your literary tendencies.
4. Receive:

   * A roast
   * A personality archetype
   * Strengths and weaknesses
   * Compatibility warnings
   * New book recommendations
